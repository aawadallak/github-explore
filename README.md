### ğŸš€  How to execute the project? 

Before start, you will need to have on your machine the following tools:
[Git](https://git-scm.com), [Node.js](https://nodejs.org/en/). 
Furthermore is good have some code editor as vscode [VSCode](https://code.visualstudio.com/)

```bash

# Clone this repository
$ git clone git@github.com:aawadallak/GitHub-Explore

# Acess the project folder in your terminal
$ cd GitHub-Explore

# Install all dependecies
$ yarn install

# Run the applicaton in develop mode
$ yarn start

# The application will run on port:3000 - acess http://localhost:3000  
  
Enjoy!
```
